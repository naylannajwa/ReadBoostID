# ReadBoostID
ReadBoost ID adalah aplikasi mobile berbasis Android yang dirancang untuk meningkatkan minat dan kemampuan literasi masyarakat Indonesia melalui pengalaman membaca yang lebih menarik, adaptif, dan interaktif. Aplikasi ini menggabungkan konten bacaan, gamifikasi, AI reading assistant, serta sistem CRUD untuk admin dalam mengelola konten literasi.
